# Keyboard Shortcuts

This document lists all keyboard shortcuts available in the imxup GUI.

## Tab Management

| Shortcut | Action |
|----------|--------|
| **Ctrl+T** | Create new tab |
| **Ctrl+W** | Close current tab (except Main tab) |
| **Ctrl+Tab** | Switch to next tab |
| **Ctrl+Shift+Tab** | Switch to previous tab |

## Gallery Management

| Shortcut | Action |
|----------|--------|
| **Delete** | Remove selected galleries from queue |
| **Ctrl+C** | Copy BBCode for selected galleries |
| **F2** | Rename selected gallery |
| **Home** | Jump to first visible gallery in current tab |
| **End** | Jump to last visible gallery in current tab |

## Application & UI

| Shortcut | Action |
|----------|--------|
| **Ctrl+,** | Open Comprehensive Settings |
| **Ctrl+.** | Show keyboard shortcuts help |
| **Ctrl+R** | Toggle right panel visibility |
| **F1** | Open help dialog |

## Advanced Features

### Gallery Table
- **Ctrl+Click** - Select multiple galleries
- **Shift+Click** - Select range of galleries
- **Enter** (in editable fields) - Save and move to next row (Custom Fields or External App Fields)

### Context Menu (Right-Click)
While not keyboard shortcuts, these frequently used actions are available via right-click on selected galleries:

- **Copy BBCode** - Copy formatted BBCode for selected galleries
- **Open Gallery Link** - Open gallery URLs in browser
- **Regenerate BBCode** - Regenerate BBCode with current template
- **Move to...** - Move selected galleries to another tab
- **View BBCode** - View full BBCode in a dialog

## Log Viewer

| Shortcut | Action |
|----------|--------|
| **Ctrl+C** (in log) | Copy selected log entries to clipboard |

## Tips

- **Drag & Drop**: Drag image folders directly onto the application to add them to the queue
- **Tab Renaming**: Double-click any tab name to rename it
- **Filtered Rows**: Home/End keys respect hidden rows (filtered galleries)
- **Keyboard Navigation**: Use Tab key to navigate between fields in settings dialogs

## Configuration Tips

- Use **Ctrl+,** frequently to access settings for credentials, templates, and file hosts
- **F1** provides context-sensitive help throughout the application
- Check the status bar at the bottom for quick shortcut hints
