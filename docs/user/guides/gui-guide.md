# Using the Interface

A quick tour of the IMXuploader window.

---

## Main Areas

### Queue Table (Center)

Shows all your galleries with:
- **Name** - Gallery name (right-click to rename)
- **Status** - Current state (Ready, Uploading, Completed, Failed)
- **Progress** - Upload percentage and speed
- **Images** - Number of images in the gallery
- **Size** - Total file size

**Right-click any gallery** for options like Copy BBCode, Retry, Move to Tab, or Remove.

### Toolbar (Top)

| Button | What It Does |
|--------|--------------|
| **Add Folder** | Browse for folders to add |
| **Start All** | Begin uploading all ready galleries |
| **Pause All** | Pause all active uploads |
| **Clear Completed** | Remove finished galleries from the queue |
| **Settings** | Open the settings dialog |

### Right Panel

- **Quick Settings** - Thumbnail size, template selection
- **Log Output** - Real-time upload messages

### Tabs (Bottom)

Organize galleries into groups. Right-click the tab bar to create new tabs.

---

## Gallery Status

| Status | Meaning |
|--------|---------|
| **Ready** | Scanned and ready to upload |
| **Uploading** | Currently uploading |
| **Paused** | Paused by you |
| **Completed** | Upload finished successfully |
| **Failed** | Upload failed - check status column for error |

---

## Managing Galleries

### Add Galleries

- **Drag and drop** folders or archives onto the queue
- Click **Add Folder** to browse
- Drop ZIP, RAR, or 7Z archives (automatically extracted)

### Remove Galleries

- Select galleries and press `Delete`
- Or right-click and select **Remove**
- Use **Clear Completed** to remove all finished uploads

### Move Between Tabs

- Right-click a gallery and select **Move to Tab**
- Or drag galleries to another tab

---

## Settings Overview

Click **Settings** in the toolbar to open the settings dialog.

| Tab | What You'll Find |
|-----|------------------|
| **General** | Thumbnail size, retry count, concurrent uploads |
| **Credentials** | imx.to username and password |
| **Templates** | Create and edit BBCode templates |
| **File Hosts** | Configure Rapidgator, Keep2Share, etc. |
| **Advanced** | Timeouts, logging, storage paths |

---

## Tips

- **Minimize to tray** - The app keeps running in the background
- **Theme toggle** - Switch between dark and light modes in the toolbar
- **Archives** - Just drop ZIP/RAR files like regular folders
- **Multiple instances** - Opening another window adds galleries to the existing one
