# Keyboard Shortcuts

This document lists all keyboard shortcuts available in the imxup GUI.

## Tab Management

| Shortcut | Action |
|----------|--------|
| **Ctrl+T** | Create new tab |
| **Ctrl+W** | Close current tab (except Main tab) |
| **Ctrl+Tab** | Switch to next tab |
| **Ctrl+Shift+Tab** | Switch to previous tab |

## Gallery Management

| Shortcut | Action |
|----------|--------|
| **Delete** | Remove selected galleries from queue |
| **Ctrl+C** | Copy BBCode for selected completed galleries |
| **F2** | Rename selected gallery |


## Application

| Shortcut | Action |
|----------|--------|
| **Ctrl+,** | Open Comprehensive Settings |
| **Ctrl+.** | Show this keyboard shortcuts help |

## Tips

- **Multiple Selection**: Use Ctrl+click or Shift+click to select multiple galleries
- **Drag & Drop**: Drag image folders directly onto the application to add them
- **Context Menu**: Right-click on galleries for additional options like "Copy BBCode", "Open Gallery Link", and "Regenerate BBCode"
- **Tab Operations**: Double-click tab names to rename them

## Context Menu Actions

While not keyboard shortcuts, these frequently used actions are available via right-click:

- **Copy BBCode** - Copy BBCode for selected galleries
- **Open Gallery Link** - Open gallery URLs in browser
- **Regenerate BBCode** - Regenerate BBCode with current template (works for multiple selections)
- **Move to...** - Move galleries between tabs
- **View BBCode** - View full BBCode in dialog

For more help, see the other tabs in this help dialog.
