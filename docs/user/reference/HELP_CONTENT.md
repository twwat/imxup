# Welcome to IMXuploader

Upload image galleries to imx.to and premium file hosts with one click.

---

## Quick Start

**Add galleries** - Drag folders or archives onto the queue, or click **Add Folder**

**Upload** - Click **Start All** or press `Ctrl+U`

**Get your links** - Right-click a completed gallery and select **Copy BBCode**

## The Main Window

| Area | What It Does |
|------|--------------|
| **Queue Table** | Shows all your galleries and their upload status |
| **Toolbar** | Add folders, start uploads, open settings |
| **Progress Bar** | Current upload progress and speed |
| **Tabs** | Organize galleries into groups |

---

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl+U` | Start uploads |
| `Ctrl+C` | Copy BBCode |
| `Delete` | Remove selected gallery |
| `F2` | Rename gallery |
| `Ctrl+Tab` | Next tab |

See [All Shortcuts](getting-started/keyboard-shortcuts.md) for the complete list.

## Need Help?

- **Upload failed?** Check the error message in the Status column, then retry
- **Slow uploads?** Your internet speed is the limit, not the app
- **Missing credentials?** Open **Settings** and check the **Credentials** section

Still stuck? Check [Troubleshooting](troubleshooting/troubleshooting.md).
